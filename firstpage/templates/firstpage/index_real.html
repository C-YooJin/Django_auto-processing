Hi, This is CDP

Clova Data auto-Processing!

When you model artificial intelligence, data is one of the most important thing. Without data, AI models cannot be created. CDP is the first step before AI modeling. CDP offers several crawlers for collect raw data. You can get the image data itself and the JSON file containing the meta informations. CDP will continue to improve! If you have any questions, please contact Data and connection team.

What we support!

Google Crawler

We offer Google image crawlers. It's based on the Python library, icrawler. You can crawl images from google with json files including eta information or image only.

Flickr Crawler

We provide a flickr image crawler. Flickr is an online photo-sharing community. We provide json files including url, auto, license information, pixel size, etc.

Instagram Crawler

We offer Instagram crawlers. You can search for the image you want based on a hashtag. You can select the download type between Image only and json types.
